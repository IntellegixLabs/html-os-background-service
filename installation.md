# Installation Guide

## Prerequisites

- Node.js
- npm

## Steps

1. Clone the repository:
    ```sh
    git clone <repository-url>
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Start the development server:
    ```sh
    npm start
    ```

4. Open your browser and navigate to `http://localhost:3000`.
